# Jogos Escolares

Sistema para gestão de jogos escolares e competições esportivas em ambiente educacional.

## Sobre o Projeto

Este repositório visa centralizar o desenvolvimento de uma solução para organização, cadastro e acompanhamento de jogos e competições escolares.

## Como Contribuir

1. Clone o repositório
2. Crie uma branch para sua feature (`git checkout -b feature/nova-funcionalidade`)
3. Commit suas alterações (`git commit -m 'Adiciona nova funcionalidade'`)
4. Envie para a branch (`git push origin feature/nova-funcionalidade`)
5. Abra um Pull Request
